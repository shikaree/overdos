Bordello

 Bordello is the ųverDOS version of Othello/Reversi. 
 You are 'o' and give the coordinates for next move.
 Passing is done by giving the coordinates <space> <space> and
 you may quit by giving the coordinates 'q' 'q'.

 At level 60, Bordello is VERY unlikely to lose.
 


Bõrs

 Bõrs is the basic command interpreter. It supports the following commands:
 * clear		            - Clears the screen.
 * kill <PIN>			    - Kills a process using its pin.
 * get_priority <PIN>		    - Returns the priority of a process.
 * pez				    - Lists all active processes.
 * reboot			    - Reboots the system and returns control 
				      to PMON.
 * dõcka			    - Creates an exception (does an illegal 
				      memory reference).
 * ls			            - Lists all runnable applications.

 If the command wasn't recognized Bõrs tries to start an application
 using the string given.



Nibbles
 
 Keys: i,j,k,l and q (quit).



Phil

 Is an implementation of the dining philosophers problem.
 It creates 5 subprocesses (phil_th) who communicate through 5 mailboxes
 and send their output to the main process which in turn outputs it
 to the screen.



Prodcons

 Example of a producer/consumer program. The main process takes the role
 of the producer which creates a subprocess which becomes the consumer.



Tetris

 Keys: j,k,l,space (drop) and q (quit).
 
       

Tic

 An example of a Spim program running on ųverDOS. Coordinates are given on the
 form [row][column]. There is no way to quit until the game is over.
 Running multiple instances of Tic is not recommended.



Time

 Prints the date and time.



Top

 Should be considered a part of the kernel.
 It is the only application directly accessing the pcb- and mailbox-lists.
 It shows the number of processes, which of them that are running or
 blocked and information about them such as PIN, parent PIN, status,
 name, priority, mailbox usage, stack- and CPU usage.
 Top also shows the uptime.

 The information is updated once every 2 seconds and in between top
 blocks itself, prohibiting it from taking more CPU than necessary.

 Any key kan be used to exit top.



Setdate

 To change the date and time. Must be given in two-digit format.
 In this order: year:month:day:hour:min:sec



Visky

 Visky is an extended command interpreter. It has the following commands:

 * bomb <name> <prior>              - Starts as many processes as possible.
 * clear                            - Clears and resets the screen.
 * dõcka                            - Hang Visky (traditional).
 * get date                         - Shows current date and time.
 * get priority <PIN>               - Get priority of process with PIN.
 * kill <PIN>                       - Kills process with specified PIN.
 * killall <name>                   - Kills all processes matching name.
 * ls                               - Show program listing.%
 * pez                              - Show active processes.
 * reboot                           - Reboots the system.
 * set priority <PIN> <prior>       - Change priority for process with PIN.
 * set date <Y> <M> <D> <H> <M> <S> - Set current date and time.      

 Other features include using the command help (to get the above information),
 and tab-completion.
