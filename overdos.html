ÖverDOS — Quick Guide

ÖverDOS is a small multitasking operating system for the Algorithmics P4032 MIPS computer, written in 2002 as a student project. This page runs it unchanged on a JavaScript emulator.

Getting started

The system boots straight into Visky, the command shell (prompt visky@överDOS>). Type a command and press Enter. Good first commands: help (all commands), ls (list programs), pez (running processes).

Running programs

Just type a program's name and press Enter. Some take over the whole screen (the games); others print something and drop you back at the prompt.

Two keys worth knowing

  • ` (back-tick) — Focus switch. The process “in focus” is the one that receives your keystrokes. Press ` and then space (or any key) to cycle focus to the next running process — the switch is applied on the following keystroke, so the back-tick alone won't do anything until you press again.
  • ' (apostrophe) — New shell. Starts a fresh Visky. Handy if you accidentally close or hang your command interpreter.

Things to try

  • tetris — move j/l, rotate k, drop space, quit q
  • nibbles — snake; steer with i/j/k/l, quit q
  • tic — tic-tac-toe (1 or 2 players)
  • top — live process monitor  ·  phil, prodcons — concurrency demos
  • bärs — a second, minimalist command shell
  • get date — show the current date and time

About

The original emulator — the program that emulates the Algorithmics P4032 hardware and runs this OS — was written by Andreas Westling. This page is a faithful JavaScript port of that emulator, so ÖverDOS boots here unchanged.

ÖverDOS itself supports full VT100/ANSI terminal emulation (hence the colours) and partial SPIM compatibility. The OS was written by Juan Pablo Fernández Álvarez, Johannes Holmberg, Magnus Jansson, Martin Kjellin, Jesper Lundgren and Andreas Westling (2002).
